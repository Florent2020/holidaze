# 🌍 Holidaze

Holidaze is a modern accommodation booking platform built with React and powered by a WordPress REST API.  
The project allows users to browse accommodations, view details, and send booking enquiries, while admins can manage listings and messages.

---

## 🚀 Features

- 🔍 Browse and search accommodations
- 🏨 View detailed accommodation pages
- 📅 Send booking enquiries
- 📩 Contact form with backend integration
- 🔐 Admin panel (login required)
- 🧑‍💼 Manage accommodations (add/edit/delete)
- 📬 View messages and enquiries
- 🌙 Dark mode support
